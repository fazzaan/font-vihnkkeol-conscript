### Version 0.500 info

Version 0.500 is the first major release in 2 months.
It contains a few new letters.
When choosing the font file from the .zip archive:
* type 1 uses the original font name (Q1Vnkl) which was intended to be a testing name;
* type 2 uses a new, proper name (Vihnkkoel Rune).

If you install type 1, it will overwrite the old font if you already have it installed. This will force-update the font in all documents and files that you've already created.

If you install type 2, it will install the new font _alongside_ the old font. This means that you'll be able to compare the old and new fonts -- but it also means that you'll have to manually change the font in every document and files that you've already created if you want them to use the new font.
